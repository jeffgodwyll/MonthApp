'DOCTYPE' keyword is not case sensitive. So, or , both will do.
As soon a browser finds in the starting of an HTML document, it represents the document in standard mode. If you don't use doctype in the starting of an HTML document, browser goes to the quirky mode. In that case, you may find that certain content is not being displayed the way you marked them up. So, it is a good practice to start your HTML document with a doctype.
Even if you don't use any other new HTML5 tag or feature, since the browser goes to the standard mode, you can use the doctype of html5.
